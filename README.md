# Panduan

## Langkah-langkah

1. Download ZIP file dari Github.
2. Extract file.
3. Buka file di Visual Studio Code (VSCode) untuk melihat kode.
4. Buka XAMPP dan nyalakan Apache serta MySQL.
5. Pergi ke phpMyAdmin dan import file SQL dari repository Github.
6. Buka browser dan akses `localhost/index.php`.
